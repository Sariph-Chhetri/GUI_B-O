This is a plain HTML/CSS/JS project — no framework, no Node.js, no build step needed.

To run it, just open the HTML file directly:

Navigate to the dashboard-project folder
Double-click dashboard.html in your file explorer

It will open straight in your browser. You can do the same for invoices.html and team.html.
Alternatively, if you're using VS Code, install the Live Server extension, right-click dashboard.html and choose "Open with Live Server" — this is nicer because it auto-refreshes when files change.
